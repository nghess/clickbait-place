# clickbait_dry_morris_maze

Morris maze task; captures Pylon camera video, flips/grayscales/thresholds the image, tracks mouse centroid, and overlays a virtual maze grid with target distribution visualization.

## Files

- `clickbait_dry_morris_maze.bonsai` — Bonsai-rx workflow
- `clickbait_dry_morris_maze.bonsai.layout` — Bonsai layout file
