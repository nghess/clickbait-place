# clickbait_behavior_3port

Three-port behavior task with DAQmx analog input for sniff/poke sensors, PointGrey camera for video, and a grid-based clickbait task with maze coordinates and reward delivery logic.

## Files

- `clickbait_behavior_3port.bonsai` — Bonsai-rx workflow
- `clickbait_behavior_3port.bonsai.layout` — Bonsai layout file
- `clickbait_behavior_3port_task.py` — Extracted Clickbait Task IronPython script
