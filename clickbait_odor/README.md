# clickbait_odor

Odor-cued spatial task; grid-based target generation with standard normal distribution (no log-normal, no flip states). Captures video and tracks centroid for reward delivery via two water ports.

## Files

- `clickbait_odor.bonsai` — Bonsai-rx workflow
- `clickbait_odor.bonsai.layout` — Bonsai layout file
- `clickbait_odor_task.py` — Extracted Clickbait Task IronPython script
