# clickbait_anemo

Records behavior session using Pylon camera; converts to grayscale, applies Gaussian smoothing and binary thresholding, detects mouse centroid via contour finding. Includes a session timer that auto-stops the workflow.

## Files

- `clickbait_anemo.bonsai` — Bonsai-rx workflow
- `clickbait_anemo.bonsai.layout` — Bonsai layout file
