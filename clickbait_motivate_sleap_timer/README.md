# clickbait_motivate_sleap_timer

SLEAP-based task with a 5-second trial timeout and incorrect-trial counter; uses a smaller 3x7 grid, minimum 1 trial before flip-state transitions.

## Files

- `clickbait_motivate_sleap_timer.bonsai` — Bonsai-rx workflow
- `clickbait_motivate_sleap_timer.bonsai.layout` — Bonsai layout file
- `clickbait_motivate_sleap_timer_task.py` — Extracted Clickbait Task IronPython script
