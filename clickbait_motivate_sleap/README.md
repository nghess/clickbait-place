# clickbait_motivate_sleap

SLEAP pose-estimation version of clickbait_motivate; adds NaN handling for centroid coordinates from pose tracking, uses log-normal distributions with a minimum of 2 trials before flip-state transitions.

## Files

- `clickbait_motivate_sleap.bonsai` — Bonsai-rx workflow
- `clickbait_motivate_sleap.bonsai.layout` — Bonsai layout file
- `clickbait_motivate_sleap_task.py` — Extracted Clickbait Task IronPython script
