# clickbait_motivate

Enhanced motivation task with 2D (normal or log-normal) target distribution, flip-state transitions, trial counting, ITI management, and reward delivery via left/right water ports.

## Files

- `clickbait_motivate.bonsai` — Bonsai-rx workflow
- `clickbait_motivate.bonsai.layout` — Bonsai layout file
- `clickbait_motivate_task.py` — Extracted Clickbait Task IronPython script
